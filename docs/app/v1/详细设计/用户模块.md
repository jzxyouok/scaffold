# 用户登陆模块
## 注册 - 获取手机号验证码
**请求URL：** 
- `/member/user/regist-phone-code`

**请求方式：**
- GET

**参数：** 

|参数名|必选|类型|说明|
|:---  |:---|:---|:---|
|phone|是|string|手机号|
|timestamp|是|string|时间戳|

**返回示例**
```
    {
        "success": true,
        "data": {
            "msg": "发送成功"
        },
        "time_point": 1528167092
    }
```
